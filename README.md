# SpringHibernateDB
WebApps using different technologies, in common Spring and Hibernate.

-SpringHibernateDB:
  *Spring framework
  *Hibernate
  *JPA validation
  *MySQL Server database
  *JSP
  *Bootstrap & Hover
  *Maven build
-RESTcorporativeDB:
  *RESTful application
  *Spring framework
  *Hibernate
  *MySQL Server database
  *HTML, JavaScript & jQuery (No JSP)
  *Bootstrap & Hover
  *Spring-boot application
  *Maven build
-RESTandANGULAR:
  *RESTful application
  *Spring framework
  *Hibernate
  *JPA validation
  *MySQL Server database
  *ANGULAR JS (HTML & JS)
  *Bootstrap & Hover
  *Spring-boot application
  *Maven build
